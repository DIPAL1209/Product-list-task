-Product List task

This is a simple full-stack Product List task built using React (frontend) and Node.js with Express (backend).


-What the project does

Displays a list of products
Allows search, category filter, and price sorting
Fetches data from backend APIs

-Frontend

Built using React
Used useState and useEffect
Connected to backend using Axios
Search works only after clicking the Search button
UI updates automatically when data changes

-Backend

Built using Node.js and Express
Created APIs to fetch products
Implemented search, filter, and sort logic
Sends structured JSON responses

-Tech Stack

Frontend: React, Axios, MUI, CSS
Backend: Node.js, Express, Database

-How to run
# Backend
npm install
npm run dev

# Frontend
npm install
npm run dev
